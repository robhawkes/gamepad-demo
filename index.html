About

This demo has been created as a visual representation of the JavaScript Gamepad API that has been introduced to both Firefox and Chrome. Right now the demo requires a special build of Firefox Nightly to work. Chrome support will be added soon.

Rob Hawkes is the author of this demo. He is a Technical Evangelist at Mozilla who focusses on game development on the Web.

Follow @robhawkes

Known issues

There are a few bugs and aspects of this demo that are known not to work:

  • Requires a custom build of Firefox
  • No support for Chrome
  • Only works with xBox 360 and PS3 gamepads
  • No automated discovery of gamepad type
